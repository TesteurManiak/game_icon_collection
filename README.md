# game_icon_collection

A collection of game icons extracted from rom files. The icons are organized by platform and game title.

You can contribute to this collection by submitting icons from your own rom files. Check the instructions in the `contributing/` folder for details on how to extract icons for different platforms and how to submit them.

You can download the entire collection as a zip file from the releases section, or you can browse the icons online in the `icons/` folder.
